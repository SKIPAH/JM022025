### Installation

1. Clone the repository
   ```sh
   git clone https://github.com/SKIPAH/JM022025.git
   ```
2. Locate the project directory
   ```sh
   cd project directory
   ```
3. Install packages
    ```sh
   npm install
   ```
4. Run the project on browser localhost
   ```sh
   npm run dev
   ```
